Simple Flask Form

Description
This project is a simple form handling web application built using Flask. It includes form validation and integrates with an SQLite database to store user inputs.

Features
- User input form with validation
- Backend handling using Flask
- SQLite database integration
- Minimal dependencies for easy setup

Technologies Used
- Flask (Python Web Framework)
- SQLite (Database)
- HTML, CSS (Bootstrap) (Frontend)

Installation
1. Clone the repository:
   bash
   git clone https://github.com/yourusername/form-handling-project.git
   
2. Navigate to the project directory:
   bash 
   cd Simple-Flask-Form
   
3. Create a virtual environment and activate it:
   bash
   python -m venv venv
   source venv/bin/activate   On Windows use `venv\Scripts\activate`
   
4. Install dependencies:
   bash
   pip install -r requirements.txt
   

Usage
1. Run the Flask application:
   bash
   python app.py
   
2. Open your browser and go to `http://127.0.0.1:5000/`

Database Setup
The application uses SQLite as the database. The schema is automatically created when running the application for the first time.
